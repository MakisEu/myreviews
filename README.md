Για το GUI του Χρήστη(User) έχουμε 2 καρτέλες.

Στην πρώτη Dashboard υπάρχει μια λίστα που περιέχει όλα τα καταλύματα που έχουν αξιολογήσει εμφανίζοντας το 
όνομα, τον τύπο, την τοποθεσία και τον βαθμό αξιολόγησης που έχουν δώσει και τον μέσο όρο της βαθμολογίας που έχουν δώσει συνολικά στα καταλύματα που
έχουν αξιολογήσει. Επιλέγοντας μια αξιολόγηση τους εμφανίζει τα όλα χαρακτηριστικά του καταλύματος. Το κουμπί Refresh list ανανεώνει τη λίστα με τα καινούρια αποτελέσματα.

Στη δεύτερη καρτέλα Search ο χρήστης δίνει πόλη τύπο όνομα και παροχές ενός καταλύματος και του επιστρέφεται σε μία λίστα το όνομα,
ο τύπος, η τοποθεσία και ο μέσος όρος των αξιολογήσεων των καταλυμάτων που ταιριάζουν
στα κριτήρια αναζήτησης. Το κουμπί Refresh list ανανεώνει τη λίστα με τα καινούρια αποτελέσματα της λειτουργίας search. 
Επίσης, εισάγοντας το UID(Unique id) ενός καταλύματος, βαθμό και περιγραφή μπορεί να προσθέσει, να τροποποιήσει ή να διαγράψει μια αξιολόγηση.

Για το GUI του Παρόχου(Provider) έχουμε 2 καρτέλες.

Στην πρώτη Dashboard υπάρχει μια λίστα που περιέχει όλα τα καταλύματα που έχουν καταχωρίσει εμφανίζοντας το
όνομα, τον τύπο, την τοποθεσία και τον βαθμό αξιολόγησης που έχει το κάθε κατάλυμα, όλες τις πληροφορίες ενός καταλύματος, μαζί με το συνολικό αριθμό αξιολογήσεων και το μέσο όρο αξιολογήσεων.
Επιλέγοντας ενα κατάλυμα τους εμφανίζει τα όλα χαρακτηριστικά του καταλύματος. Το κουμπί Refresh list ανανεώνει τη λίστα με τα καινούρια αποτελέσματα.

Στη δεύτερη καρτέλα Accommodation Manager, ο πάροχος μπορεί να διαγράψει ένα κατάλυμα με συγκεκριμένο UID, να επεξεργαστεί ένα κατάλυμα με συγκεκριμένο UID με βάση τα πεδία που έχουν συμπληρωθεί,
και να προσθέσει ένα κατάλυμα συμπληρώνοντας όλα τα υποχρεωτικά πεδία. 

Προαιρετικές παροχές: Δέχεται παροχή στη μορφή property1-value1,property2-value2...
Ειδικά χαρακτηρίστηκα:
Για Διαμέρισμα, floor: ο όροφος στο οποίο είναι το διαμέρισμα, space: πόσα άτομα χωράει, guard: Αν έχει φύλακα ή όχι, elevator: αν έχει ανελκυστήρα ή όχι
Για Ξενοδοχείο, stars: πόσα αστέρια είναι, floor: πόσου ορόφους έχει, suite: Αν έχει σουίτα ή όχι.
Για Μεζονέτα, roommates: πόσοι συγκάτοικοι υπάρχουν.

Τρέξτε την main στο φάκελο src για να αρχίσετε το πρόγραμμα.
Προειδοποίηση: Αν τρέξετε τα test, τα δεδομένα στα αρχεία μπορεί να χαθούν.
