Για το GUI του Χρήστη(User) έχουμε 2 καρτέλες.

Στην πρώτη Dashboard υπάρχει μια λίστα που περιέχει όλα τα καταλύματα που έχουν αξιολογήσει εμφανίζοντας το
όνομα, τον τύπο, την τοποθεσία και τον βαθμό αξιολόγησης που έχουν δώσει και τον μέσο όρο της βαθμολογίας που έχουν δώσει συνολικά στα καταλύματα που
έχουν αξιολογήσει..Επιλέγοντας μια αξιολόγηση τους εμφανίζει τα όλα χαρακτηριστικά του καταλύματος.Το κουμπί Refresh list ανανεώνει την λίστα με τα καινούριγα αποτελέσματα της λειτουργίας search.

Στην δεύτερη καρτέλα Search ο χρήστης δίνει πόλη τύπο όνομα και παροχές ενός καταλύματος και του επιστρέφεται σε μία λίστα το όνομα,
ο τύπος, η τοποθεσία και ο μέσος όρος των αξιολογήσεων των καταλυμάτων που ταιριάζουν
στα κριτήρια αναζήτησης.Επίσης εισάγωντας ένα όνομα καταλύματος, βαθμό και περιγραφή μπορεί να προσθέσει, να τροποποιήσει ή να διαγράψει μια αξιολόγηση.
